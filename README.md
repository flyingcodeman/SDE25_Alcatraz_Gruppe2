# SDE25_Alcatraz_Gruppe2
DSI Projekt Alcatraz - RMI &amp; Spread

## Basic funktionality:

- Compile the java files for client and server:
**_javac Server.java ClientInterface.java Client.java_**

- Start the RMI registry on the SERVER:
_**rmiregistry**_

- Start the server:
_**java Client**_

- Start the client(s):
_**java Client**_
